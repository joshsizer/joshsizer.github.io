  • daisy_cv_android

    (Color) Vision processing android application written for Team 341, Miss Daisy. While shining a bright colored LED light at FRC game challenge targets framed by retro reflective tape, the application takes video at 30 FPS. These frames are filtered by HSV values, and then contour fit. Contours are filtered by shape, size, width-height ratio, ect. in order to filter out unwanted detections. (X,Y) pixel coordinates are sent back to the Robot control system over ADB (Android Debug Bridge) through a custom network protocol.

  • NumberLogger

    A small java library that can store an arbitrary number of rows and columns of data. It has functionality to export this data to an XML document. This library was written in order to debug Robot control logic and to tune closed loop control systems (PID, trajectories, paths, ect).
